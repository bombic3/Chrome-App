# Moment Todo List
API를 연동하여 위치와 날씨를 출력하는 To Do List 구현

## 🖥️ 프로젝트 소개
로그인 기능이 있고 위치와 날씨 값 받아 출력하여 배경화면과 명언 랜덤으로 나오는 To Do List입니다.

#### 위치/날짜
